ADTracking Fraud Detection

(Python Notebook)

Given the clicking data collected by a mobile advertisement company, this project tries to answer the question: does the click on an app AD result into download the app? By being able to answer this question, we can potentially identify fraudulent clicks. This project is based on a Kaggle competition named TalkingData AdTracking Fraud Detection Challenge.

You can check the jupyter notebook containing data preparation (including feature engineering), model training, and evaluation of machine learning models to tackle the problem.

Language Variety Identification

The identification of the language in which a text is written is an important preprocessing step in natural language processing pipelines with language-specific algorithms. To go even further on developing specialized models we can consider the variants of a pluricentric language such as the Portuguese. In this project, I show how one can build a learning algorithm to differentiate between the Brazilian and European Portuguese language varieties.

You can check the repository with the source code for training and evaluating the models.

Document Image Binarization

The document image binarization is a crucial step in pipelines involving document analysis and recognition as it provides better visualization of the desired content. In this project, I have explored the document binarization of historical handwritten texts, which are very challenging due to the background noise.

You can check the repository with the source code for training and evaluating the models.
